## Usage
We use [Helm](https://helm.sh/) to package [EFA](https://docs.aws.amazon.com/AWSEC2/latest/UserGuide/efa.html) device plugin:
```
helm install efa ./aws-efa-k8s-device-plugin -n kube-system
```
